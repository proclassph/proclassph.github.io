6. Kiko regularly runs around an Olympic size track oval measuring 400 meters. His trainer, Joy, observed that his running times are normally distributed with a standard deviation of ten seconds. A. What is Kiko's average running time around the 400-meter track oval if 33% of his runs took him less than 120.6s • B. What proportion of Kiko's running times are above 140s? • C. Below how many seconds would you find the fastest 10% of Kiko's running times?


A. To find Kiko's average running time, we need to first determine the corresponding z-score for the 33rd percentile. Using a standard normal distribution table or calculator, we find that a z-score of approximately -0.44 corresponds to the 33rd percentile.

Next, we can use the z-score formula to find Kiko's average running time:

z = (x - μ) / σ

where z is the z-score, x is the observed running time, μ is the population mean, and σ is the standard deviation.

Plugging in the values we have:

-0.44 = (120.6 - μ) / 10

Solving for μ, we get:

μ = 124.4 seconds

Therefore, Kiko's average running time around the 400-meter track oval is 124.4 seconds.

B. To find the proportion of Kiko's running times that are above 140s, we first need to find the corresponding z-score:

z = (140 - 124.4) / 10

z = 1.56

Using a standard normal distribution table or calculator, we find that the proportion of observations above a z-score of 1.56 is approximately 0.059. Therefore, approximately 5.9% of Kiko's running times are above 140s.

C. To find the running time below which we can find the fastest 10% of Kiko's running times, we need to find the z-score that corresponds to the 10th percentile:

z = invNorm(0.10)

z = -1.28

Using the same formula as in part A, we can solve for x:

-1.28 = (x - 124.4) / 10

x = 112.0 seconds

Therefore, the fastest 10% of Kiko's running times are below 112.0 seconds.
