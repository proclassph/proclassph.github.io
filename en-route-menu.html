14. There are two routes Sally can take to travel home from school. Route A is 800 m. Route B is 1 km. What is the percentage decrease in distance if Sally took route A instead of route B?


Answer:

To find the percentage decrease in distance if Sally took route A instead of route B, we first need to calculate the difference between the distances of the two routes:

1 km = 1000 m

So, the difference in distance between Route A and Route B is:

1000 m - 800 m = 200 m

Next, we need to find the percentage decrease from Route B to Route A, which can be calculated using the formula:

% decrease = (difference / original value) x 100%

In this case, the original value is the distance of Route B (1000 m), and the difference is 200 m. Substituting these values into the formula, we get:

% decrease = (200 / 1000) x 100%

% decrease = 0.2 x 100%

% decrease = 20%

Therefore, if Sally took Route A instead of Route B, she would be traveling 20% less distance.

17. what is the principal difference between hub routing and switch routing?


Answer:

The big difference between hub and switch is in the method in which frames are being delivered. In a hub, a frame is passed along or "broadcast" to every one of its ports. It doesn't matter that the frame is only destined for one port.

24. What are two advantages of static routing over dynamic routing?


Answer:

Static routing requires very little knowledge of the network for correct implementation. Static routing uses fewer router resources than dynamic routing. Static routing is relatively easy to configure for large networks. Static routing is more secure because it does not advertise over the network.

25. Why is there a need to perform mise en place for kitchen tools, utensils, equipment, and ingredients for a menu to be prepared?​


Answer:

Mise en place is a French term meaning “putting in place” and is an essential part of any professional kitchen. It involves organizing and preparing all the necessary tools, utensils, equipment, and ingredients before beginning to cook. This helps to ensure that everything is ready to go when it’s time to start cooking, which saves time and prevents mistakes. Additionally, it helps chefs stay organized and efficient while they are working in the kitchen.
