24. why the functions of cin and cout important in c++?​


The cin and cout keywords are very popular in C++. They are used for taking inputs and printing outputs, respectively.

cin is an object of the input stream and is used to take input from input streams like files, console, etc.

cout is an object of the output stream that is used to show output.

Basically, cin is an input statement while cout is an output statement. They also use different operators.
