15. ano ba ang kaibahan ng command ekonomy sa mixed ekonomy?​


Answer:

Ang kaibahan ng command at mixed economy ay ang bilang ng economy na sangkot. Ang command economy ay isang ekonomiya na umiiral. Ang mixed economy o sa ibang salita ay dual economy. Nangangahulugan ito na ang mixed economy ay nagsasangkot ng dalawang umiiral ekonomiya. Ang command economy ay ang ekonomiya na ang gobyerno ay ang nagdedesisyon para sa lahat ng aspeto at produksyon sa bansa.
