25. blockchain maintains and records data


Blockchain is a technology that is used as a digital storage system or data bank that is connected with cryptography. Its use is inseparable from Bitcoin and other Cryptocurrencies.

Even so, many other sectors can take advantage of technological developments from this Blockchain. To make it easier to understand the meaning of Blockchain, let's look at the naming side. The block means group and the chain which means chain. The meaning of naming this technology reflects how blockchain works. Where the technology utilizes computer resources to create connected blocks (chain).

Connected blocks are later used to execute a transaction. This technology is quite interesting because it is not centralized. Blockchain can run itself using computer algorithms without any particular system that regulates it.

Blockchain working system:

To put it simply, blockchain starts when a block stores new data. The system consists of two types of records, namely transactions and blocks. What is unique about this technology is that each block will contain a cryptographic hash to form a network.

The function of the cryptographic hash is to take data from the original block and convert it into a compact string. The string is used as a detection alarm if potential sabotage is found. Its decentralized nature makes this technology not have any authority with full control but is divided into each computer that has installed special software or software.

There are so-called bitcoin miners, people who set up special computers to perform mathematical calculations to open new block gaps in the blockchain. They will calculate a new hash value based on the combination of the previous cryptographic hashes. Later, when there is a transaction between an online store that provides a Bitcoin Wallet and a buyer, the transaction will be sent to the miners or bitcoin miners. That way, a new block will be created and used to store bitcoins in the transaction.
