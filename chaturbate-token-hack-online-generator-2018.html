5. There are 2 stacks of tokens on the table. If i move 5 tokens from the taller pile to the shorter one, there would be the same number of tokens each pile. How many more original number of tokens are there in the tall pile than in the shorter one? Thank you for answering :). And can you please explain thanks!


Answer w/ Explanation:

To solve this problem, we can use algebra. Let's say that the number of tokens in the taller pile is x, and the number of tokens in the shorter pile is y. After moving 5 tokens, the taller pile will have x - 5 tokens, and the shorter pile will have y + 5 tokens. Since the number of tokens in each pile is now the same, we can write an equation:

x - 5 = y + 5

Solving for x, we can add 5 to both sides:

x = y + 10

So, the original number of tokens in the taller pile is y + 10 more than the original number of tokens in the shorter pile.
