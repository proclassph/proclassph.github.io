3. ano ang ibig meaning ng reference point sa tagalog​


Answer:

Ang "reference point" ay maaaring maipaliwanag sa Tagalog bilang "talaksan ng sanggunian" o "patunay na pinanggagalingan". Ito ay isang point of comparison o isang punto ng paghahambing kung saan sinusukat ang posisyon, layo, o direksyon ng isang bagay o pangyayari. Sa kadalasang paggamit nito, ang reference point ay ginagamit upang masukat ang paggalaw o kinakatigan ng isang bagay sa relasyon sa isang tiyak na punto.
