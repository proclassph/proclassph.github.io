4. A lottery game has balls numbered 1 through 21. What is the probability of selecting an odd number ball greater than 6?​


There are two parts to this question:

1. How many odd number balls greater than 6 are there?

2. What is the total number of balls in the lottery game?

3. There are 8 odd number balls greater than 6: 7, 9, 11, 13, 15, 17, 19, and 21.

4. The total number of balls in the lottery game is 21 (since the balls are numbered 1 through 21).

Therefore, the probability of selecting an odd number ball greater than 6 is:

number of odd number balls greater than 6 / total number of balls

= 8 / 21

= 0.38 (rounded to two decimal places)

So the probability of selecting an odd number ball greater than 6 is approximately 0.38 or 38%.

14. How many combinations of 6 numbers (no repitition) can you make out of the numbers from 1-42 to win the lottery? ​


Answer:

To calculate the number of combinations of 6 non-repeating numbers that can be chosen from a set of 42 numbers, we can use the formula for combinations, which is:

nCr = n! / r!(n-r)!

where n is the total number of items, r is the number of items to choose, and ! denotes the factorial operation.

Using this formula, we have:

42C6 = 42! / 6!(42-6)!

= (42 x 41 x 40 x 39 x 38 x 37) / (6 x 5 x 4 x 3 x 2 x 1)

= 5,245,786

Therefore, there are 5,245,786 possible combinations of 6 non-repeating numbers that can be chosen from a set of 42 numbers.
