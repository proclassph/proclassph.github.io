17. The International Airline Transportation Assiociation(IATA)assigns three letter codes to represent airport location.For example,MNL represents Ninoy Aquino International Airport in Manila.How many different Airport codes are possible?​


Answer:

17576 possible codes

Step-by-step explanation:

In the given problem, it is a multiple choice (combinatorics). A way of calculating possible arrangement of items.

Why it is not combination or permutation?

Because the given problem has no other rules or condition. In fact you can choose 3 letters from the 26 letters in the alphabet and repititions are allowed.

In the given problem, the solution will be:

n=26 (since it has 26 letters in the alphabet)

r= 3 (since it has 3 letters in making a code)

In the problem above, there has no other condition, repitition of the letters is permitted.

You can solve the problem in this way,

____, ____, _____

26, 26, 26 (in each line, you can use the 26 letters)

Thus, 26x26x26 equals 17576 possible ways.

Other way to solve the problem is using the formula.

The formula for multiple choice is M= nMr = n^r

where:

M is the number of Multiple Choices

n is the number of items

r is the number of items taken by

M= 26^3

M= 17576 possible number of codes can be made.

Therefore, there 17576 possible codes.
