25. What is a Java operator? Give examples of operator?​


A Java operator or operator is/are used to perform operations on values or variables.There are five types of Java operators, namely:

* Arithmetic operators

* Assignment operators

* Comparison operators

* Logical operators

* Bitwise operators

Each has its own members:

Arithmetic operators are operators used for performing mathematical computations. Example of this are:

+ Addition

- Subtraction

* Multiplication

/ Division

% Modulo Or Remainder Operator

++ Increment or Add the value by 1

-- Decrement or Subtract the value by 1

Assignment operators are used to assign the variable a value, or to define a variable.Examples of this are:

= Assignment or change the value

+= Addition assignment

-= Subtraction assignment

*= Multiplication assignment

/= Division assignment

%= Modular assignment

...

Comparison operators are operators used for comparing 2 values.Examples of this are:

== Equality operator

!= Inequality operator

>= Greater than operator

<= Lesser than operator

> Greater than operator

< Lesser than operator

Logical Operators or Boolean Logical Operators are used to determine the logic between two Booleans or values.

Examples of this are:

&& Logical and

|| Logical Or

! Logical Not

Bitwise operators are used for binary logic or bits of a number.Examples of this are:

& Bitwise and

| Bitwise or

~ Bitwise not

^ Bitwise xor

<< Zero fill left shift

>> Signed fill right shift

>>> Zero fill right shift
